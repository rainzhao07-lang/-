# 本命猫 · 女性视角模拟用户调研（推演版 V1）

> **方法说明**：本文不是真实问卷数据，是基于目标人群（18–30 岁、小红书/抖音活跃、女性为主）**公开的行为规律与消费心理**做的角色扮演推演。用途是替代"拍脑袋"，给内容与体验优化提供有依据的方向。真实数据要等上线后的漏斗埋点（答题完成率、分享卡保存率、付费转化率）来校准——本文所有结论都应被真实数据推翻或确认。

## 二、全流程走查（她们会在哪一步心动 / 出戏 / 流失）

**① 落地页**
- 阿檬/Yuki：第一眼看的是"好不好看、值不值得晒"。当前奶油底 + 暖光 + 浮动猫 → ✅ 及格偏上。
- 然然：会找"要花多少钱"，价格徽章前置 → ✅ 无被骗感。
- ⚠️ 共性：标题「测测你内心住着哪只猫」够钩，但副标题偏理性（"冷静判断"）。对阿檬这类玩票用户，情绪钩子可以更"心动"一点。

**② 答题（16 题）**
- 小满/然然：场景化改写（"给自己 vs 给毛孩子买东西""半夜急诊你的第一反应"）→ ✅ 有画面、能自我投射，不像查户口。
- ⚠️ 阿檬：16 题对纯玩票用户偏长，**第 8–10 题是流失高危区**。微反馈气泡（第 4/8/12 题）是留人关键，但目前每人格只有 2 条语料，**多测几次会重复、露馅**。
- ⚠️ 小鹿：q7「可能有人强烈反对」、q11「我可能真的扛不住突发开销」——这两个诚实选项本身没问题，但**选完之后如果结果页直接甩现实提示，她会觉得"我就知道我不配"**。

**③ 免费结果页**
- 全员：称号 + 判词 + 本命猫揭晓 → ✅ 情绪高点，是分享卡的动力来源。
- Yuki：判词要"准到起鸡皮疙瘩"才会截图。当前判词质量 OK，但**缺一段更长、更私人的"被看见"文字**——这正是付费钩子该在的位置。
- ⚠️ 小鹿：现实提示卡片文案「现实里更稳的搭档也许是 X」——措辞已经比较温柔，但对社恐/低自尊用户，**建议再加一句明确的"这不是否定你"**。

**④ 付费墙（9.9）**
- 然然：为"花费分项 + 健康雷点 + 新手清单"付费，理由成立。
- Yuki/小满：为"那段更深的、只属于我的剖析"付费。**目前 teaser 只露 freeTeaser 一句，说服力还差一口气**。
- 阿檬：大概率不付，但会晒免费卡 → 符合"自动分层"设计，不用强留。

**⑤ 报告 + 二次分享**
- 彩蛋"AI 给猫起的 3 个名字" → ✅ 超预期，是二次晒图的触发点。
- ⚠️ 报告目前全靠 LLM 现场发挥，质量会波动，9.9 的"值感"不稳。

## 三、共性发现（4 条）

1. **分享卡是增长命根子，但它的"晒得出手"取决于视觉高级感**——阿檬这类节点用户不会晒一张普通测试结果图。卡片值得单独投入。
2. **"被看见"的情绪爆点决定付费**，而现在这一段是 LLM 临场发挥、不稳定。需要每人格一段人工精修的高潮段兜底。
3. **16 题的中段是流失高危区**，微反馈是唯一的留人手，但语料太少会重复穿帮。
4. **现实条件题是双刃剑**：对然然是"专业可信"，对小鹿是"被否定"。同样的冲突提示，措辞的温度决定她是转化还是流失。

## 五、一句话结论

产品的骨架（免费给体面、付费卖升维、反冲动立意）是对的；下一步的优化重心不该是"加功能"，而是**把两处情绪浓度做满**——① 分享卡要美到值得晒，② 报告里要有一段准到让人截图的"被看见"。这两点直接对应增长飞轮的两个燃料口。其余现实条件题只需在措辞温度上对"低自尊用户"再温柔一档，避免把靶心用户（正在犹豫要不要养猫的人）在结果页当场劝退成"我不配"。
